INTRODUCTION
=============

This directory contains the Oxygen Publishing Engine version 25.0, a software solution to automate the transformations
 of DITA content into WebHelp, PDF, EPub, and plain HTML output, outside of Oxygen XML Editor (e.g from a command line).
